## 💡 使用建議

### 對於終端用戶
1. 從 [用戶指南](./USER_GUIDE.md) 開始
2. 查看主 [README](../README.md) 了解功能概覽
3. 遇到問題時參考用戶指南的故障排除部分

### 對於開發者
1. 閱讀 [架構文檔](./ARCHITECTURE.md) 了解系統設計
2. 查看 [API 文檔](./API.md) 了解後端接口
3. 參考 [設計系統](./DESIGN_SYSTEM.md) 保持 UI 一致性
4. 遵循 [貢獻指南](./CONTRIBUTING.md) 提交代碼

### 對於部署人員
1. 查看主 [README](../README.md) 的部署部分
2. 閱讀 [部署指南](../project/DEPLOY.md) 了解詳細步驟
3. 參考 [架構文檔](./ARCHITECTURE.md) 了解構建配置
