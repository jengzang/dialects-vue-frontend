- Praat - 音頻分析
- 數據工具 - 字表驗證、合併、轉換

## 音韻查詢系統

### 查字

**步驟:**
1. 導航到 `/menu?tab=query`
2. 選擇 **"查字"** 標籤
3. 輸入漢字（例如：`你好世界`）
4. 選擇地點或地區
5. 點擊 **"運行查詢"**
6. 在 **"結果"** 標籤查看結果

**示例:**
```
字符：你好
地點：廣州、深圳、香港
→ 返回「你好」在這三個地點的讀音
```

### 查中古

**步驟:**
1. 選擇 **"查中古"** 標籤
2. 構建中古音路徑：
   - **聲母** - 從下拉列表選擇（幫、端、知等）
   - **韻母** - 從下拉列表選擇（東、冬、江等）
   - **調類** - 從下拉列表選擇（平、上、去、入）
3. 選擇要顯示的特徵
4. 選擇地點或地區
5. 點擊 **"運行查詢"**

**示例:**
```
路徑：幫_東_平
特徵：聲母、韻母
地點：廣州
→ 返回該中古音在廣州的現代反映
```

### 查音位

**步驟:**
1. 選擇 **"查音位"** 標籤
2. 選擇音位組（聲母組、韻母組、調類組）
3. 輸入音位值（IPA）：例如 `p, t, k`
4. 選擇要顯示的特徵
5. 選擇地點或地區
6. 點擊 **"運行查詢"**

**示例:**
```
組別：聲母組
值：p, pʰ, m
特徵：聲母
地點：廣州、深圳
→ 返回所有帶 [p]、[pʰ] 或 [m] 聲母的字
```

### 查調

**步驟:**
1. 選擇 **"查調"** 標籤
2. 選擇地點或地區
3. 點擊 **"運行查詢"**

**示例:**
```
地點：廣州、深圳、香港
→ 返回這三個地點的完整聲調系統
```

## 地圖可視化

### 訪問地圖

1. 在查詢標籤中運行查詢
2. 導航到 **"地圖"** 標籤
3. 地圖自動加載查詢結果

### 地圖功能

**基礎地圖樣式:**
- **街道** - 默認街道地圖
- **衛星** - 衛星圖像
- **地形** - 地形圖

**控制:**
- **縮放** - 鼠標滾輪或 +/- 按鈕
- **平移** - 點擊並拖動
- **旋轉** - 右鍵並拖動（或 Ctrl+拖動）
- **傾斜** - Ctrl+垂直拖動

### 地圖模式

#### 1. 查詢模式
顯示音韻查詢的結果
- 按音韻特徵顏色編碼標記
- 密集區域自動聚類
- 點擊標記查看詳情

#### 2. 自定義模式
添加您自己的標記和註釋
1. 點擊 **"自定義"** 標籤
2. 點擊 **"添加標記"** 按鈕
3. 在地圖上點擊放置標記
4. 輸入標籤和註釋
5. 點擊 **"保存"**

#### 3. 分區模式
將地點組織成自定義區域
1. 點擊 **"分區"** 標籤
2. 點擊 **"創建分區"** 按鈕
3. 輸入區域名稱
4. 選擇要包含的地點
5. 點擊 **"保存"**

## 音韻分析工具

### 音韻統計

**訪問:** `/explore?tab=phonology`

**步驟:**
1. 輸入地點名稱
2. 點擊 **"查詢"**
3. 查看音韻矩陣

**矩陣顯示:**
- **行** - 韻母
- **列** - 聲母
- **單元格** - 聲調
- **內容** - 示例字符

### 音節統計

**訪問:** `/explore?tab=countphos`

**步驟:**
1. 輸入多個地點（逗號分隔）
2. 點擊 **"統計"**
3. 查看統計數據

**提供的統計:**
- 聲母數量
- 韻母數量
- 聲調數量
- 音節總數
- 分佈圖表

### 自定義分類

**訪問:** `/explore?tab=custom`

**步驟:**
1. 選擇特徵類型（聲母、韻母、聲調）
2. 配置分類維度
3. 選擇地點
4. 點擊 **"生成"**

## Praat 音頻分析

### 上傳音頻

**支持的格式:**
- WAV（推薦）
- MP3
- 採樣率：16kHz - 48kHz

**方法:**

1. **文件上傳**
   - 點擊 **"選擇文件"** 按鈕
   - 從計算機選擇音頻文件

2. **直接錄音**
   - 點擊 **"錄音"** 按鈕
   - 允許麥克風訪問
   - 點擊 **"開始"** 開始錄音
   - 點擊 **"停止"** 完成

### 分析設置

**音高 (F0) 設置:**
- **最小 F0** - 默認：75 Hz（男聲）
- **最大 F0** - 默認：500 Hz（女聲）

**共振峰設置:**
- **共振峰上限** - 默認：5500 Hz（女性）、5000 Hz（男性）
- **共振峰數量** - 默認：5

### 運行分析

**分析模式:**
1. **單一模式** - 將整個文件作為一個單元分析
2. **連續模式** - 分段並分別分析

**步驟:**
1. 上傳或錄製音頻
2. 配置設置
3. 選擇分析模式
4. 點擊 **"開始分析"**
5. 等待處理（進度條顯示狀態）
6. 查看結果

### 查看結果

**波形顯示:**
- 時域表示
- 縮放和平移控制

**頻譜圖:**
- 頻率-時間表示
- 共振峰疊加
- 音高輪廓疊加

**共振峰分析:**
- F1、F2、F3 隨時間變化的值
- 元音空間圖（F1 vs F2）
- 導出共振峰數據

**音高分析:**
- F0 輪廓隨時間變化
- 聲調分類
- 音高統計（平均值、範圍等）

## 數據管理

### 字表檢查

**訪問:** `/explore?tab=check`

**步驟:**
1. 上傳字表（Excel 文件）
2. 選擇要檢查的地點
3. 點擊 **"檢查"**
4. 查看驗證報告

**執行的檢查:**
- 缺失字符
- 重複條目
- 無效的 IPA 符號
- 不一致的調值

### 表格合併

**訪問:** `/explore?tab=merge`

**步驟:**
1. 上傳多個 Excel 文件
2. 選擇合併策略
3. 配置列映射
4. 點擊 **"合併"**
5. 下載合併的表格

### 粵拼轉 IPA

**訪問:** `/explore?tab=jyut2ipa`

**步驟:**
1. 輸入粵拼文本（例如：`nei5 hou2`）
2. 點擊 **"轉換"**
3. 查看 IPA 輸出（例如：`nei˩˧ hou˧˥`）

## 常見問題

**Q: 平台是免費使用的嗎？**
A: 是的，平台免費。註冊提供擴展功能。

**Q: 可以在移動設備上使用嗎？**
A: 是的，平台是響應式的，可在移動瀏覽器上使用。

**Q: 為什麼我的查詢結果為空？**
A: 檢查：
- 地點拼寫正確
- 字符存在於數據庫中
- 您有足夠的權限（註冊以獲得更多訪問權限）

**Q: 地點和地區有什麼區別？**
A: 地點是特定的方言點（例如廣州）。地區是地點組（例如粵語區）。
